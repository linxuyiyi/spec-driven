---

## 使用方法

1. **添加阻塞** 当：
   - QA 阻塞合并
   - Agent 之间对 API 合同有分歧
   - 外部依赖阻塞进度
   - 技术债务阻止实现

2. **更新状态**:
   - `pending`: 已识别，等待处理
   - `in_progress`: Agent 正在解决
   - `resolved`: 阻塞已清除

3. **移除阻塞** 当：
   - 问题已修复并验证
   - 已实现变通方案
   - 决定延期处理
