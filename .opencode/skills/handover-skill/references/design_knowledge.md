# 切换测试设计知识

## 测试维度
- 同频切换：基于 RSRP 的 A3 事件
- 异频切换：基于 RSRQ 的 A5 事件
- 异系统切换：5G→4G，4G→5G

## 关键测试点
1. **正常切换**：UE 移动触发 A3 事件，完成切换并恢复业务。
2. **切换失败**：目标小区资源不足，UE 回退源小区。
3. **切换时延**：从测量报告到切换完成的时间。
4. **条件切换**：CHO 配置与执行。
5. **双激活切换**：Make-before-break 场景。

## 边界值
- 触发时间（TTT）：0ms, 320ms, 640ms
- 偏移量（Off）：-3dB, 0dB, 3dB

## 协议依据
- 3GPP TS 38.331 (RRC)
- 3GPP TS 38.300 (移动性)
