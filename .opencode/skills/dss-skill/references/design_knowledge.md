# DSS 测试设计知识（详细版）

## 测试维度
- 功能：DSS 使能/去使能，NR UE 接入与业务承载
- 性能：资源抢占时延、吞吐量影响
- 异常：LTE 负载突增、配置错误、小区重选

## 关键测试点
1. **基础接入**：DSS 开启后，NR UE 成功附着并建立数据业务。
2. **资源抢占**：LTE PRB 占用率超过阈值（70%）时，NR 无法占用新资源。
3. **动态调整**：LTE 负载下降后，NR 自动恢复资源使用。
4. **测量报告**：基站正确下发测量配置，UE 上报 RSRP/RSRQ 包含 DSS 小区。
5. **配置错误**：DSS 频段与小区实际频段不匹配时，基站保持 LTE 正常，NR 不可用。

## 边界值
- 抢占阈值：50%, 70%, 90%
- 最小保留 PRB：1，最大分配 PRB：100

## 协议依据
- 3GPP TS 38.331 (RRC)
- 3GPP TS 38.214 (调度)
