# 测试设计技能

本技能是 5G 基站测试设计的执行层，被 TFO Agent 调用，根据专题领域执行具体测试设计。

## 定位

- **调用者**：TFO Agent
- **职责**：根据专项指导书生成测试用例

## 专题映射

| 专题 | 指导书 |
|------|--------|
| 参数约束 | references/参数约束专项指导书.md |

## 工作流程

1. **加载指导书**
   - 根据传入的专题领域，加载对应指导书

2. **分析测试对象**
   - 识别被测对象类型
   - 提取约束规则（OCL表达式或文字描述）
   - 确定测试维度（正向/负向/边界）

3. **生成用例**
   - 按指导书中的编号规范生成用例编号
   - 按格式字段逐条生成

4. **自检**
   - 编号唯一
   - 步骤完整可执行
   - 预期结果可客观判定

5. **输出**
   - 返回 Markdown 表格格式

## 输入

调用技能时需传入：
- 专题领域（如：参数约束）
- 测试需求描述
- 约束/参数定义（如有）

## 输出

Markdown 表格格式的测试用例：

| 用例编号 | 用例名称 | 级别 | 前置条件 | 测试步骤 | 预期结果 | 后置条件 |
|----------|----------|------|----------|----------|----------|----------|
