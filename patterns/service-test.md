# 业务测试 Pattern (PING/FTP)

## 触发关键词

- 中文：PING、吞吐量、FTP 上传、FTP 下载、业务测试
- 英文：PING、throughput、FTP upload、FTP download、service test

## 测试目的

验证 5G 网络建立后，UE 的数据业务功能正常，包括时延、吞吐量等关键 KPI。

## 协议参考

- 3GPP TS 38.306: UE 无线接入能力
- 3GPP TS 38.101: UE 射频要求
- ITU-R M.2410: 5G 性能指标

## 关键 KPI 要求

| 业务类型 | KPI 指标 | 要求 |
|---------|---------|------|
| PING | 时延 | < 10ms (空口理想条件) |
| PING | 成功率 | 100% |
| FTP 下载 | 峰值吞吐量 | 1.5Gbps+ (100MHz, 30kHz SCS) |
| FTP 上传 | 峰值吞吐量 | 300Mbps+ |

## 标准步骤序列 (PING 测试)

| 步骤 | 步骤名 | 操作描述 | 参数 | 预期结果 |
|------|--------|---------|------|---------|
| 1 | 环境重置 | 清空残留配置，重置测试环境 | - | 环境干净 |
| 2 | 小区建立 | 建立 NR 服务小区 | cell_id=1, BW=100MHz, SCS=30kHz, PCI=1 | 小区状态 ACTIVE |
| 3 | UE 附着 | UE 附着到服务小区 | IMSI=460001234567890, PDU=IPv4 | UE 进入 CONNECTED 态 |
| 4 | 数据通路预验证 | 验证基础数据通路 | timeout=5s | PING 测试通过 |
| 5 | 执行 PING 测试 | 向目标 IP 发送 ICMP 包 | dst_ip=8.8.8.8, count=10, timeout=1000ms | 10 个包全部发送 |
| 6 | PING 结果验证 | 检查时延和成功率 | - | 成功率 100%, 平均时延<10ms |

## 标准步骤序列 (FTP 吞吐量测试)

| 步骤 | 步骤名 | 操作描述 | 参数 | 预期结果 |
|------|--------|---------|------|---------|
| 1 | 环境重置 | 清空残留配置，重置测试环境 | - | 环境干净 |
| 2 | 小区建立 | 建立 NR 服务小区 | cell_id=1, BW=100MHz, SCS=30kHz, PCI=1 | 小区状态 ACTIVE |
| 3 | UE 附着 | UE 附着到服务小区 | IMSI=460001234567890, PDU=IPv4 | UE 进入 CONNECTED 态 |
| 4 | 执行 FTP 下载 | 从服务器下载大文件 | file_size=100MB, server_ip=192.168.1.100 | 下载完成 |
| 5 | 下载吞吐量验证 | 检查吞吐量 KPI | - | throughput > 1Gbps |

| 7 | 上传吞吐量验证 | 检查吞吐量 KPI | - | throughput > 200Mbps |

## 关键参数推荐值

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| PING 包大小 | 32 bytes | 标准 ICMP 包 |
| PING 次数 | 10 | 统计样本数 |
| PING 超时 | 1000ms | 单包超时 |
| FTP 文件大小 (下载) | 100-500MB | 大文件测吞吐 |
| FTP 文件大小 (上传) | 50-100MB | 避免 UE 缓存不足 |
| FTP 服务器 IP | 192.168.1.100 | 内网测试服务器 |

## 通过标准

### PING 测试
1. ✅ PING 成功率 100%
2. ✅ 平均时延 < 10ms (理想空口条件)
3. ✅ 无超时包

### FTP 下载测试
1. ✅ 文件传输完成
2. ✅ 平均吞吐量 > 1Gbps (100MHz 带宽)
3. ✅ 无 TCP 重传或重传率 < 1%

### FTP 上传测试
1. ✅ 文件传输完成
2. ✅ 平均吞吐量 > 200Mbps
3. ✅ UE 发射功率在合理范围

## 常见失败原因

| 现象 | 可能原因 | 排查方法 |
|------|---------|---------|
| PING 超时 | 数据通路未建立 | 检查 PDU Session 状态 |
| PING 时延大 | 空口条件差/干扰 | 检查 RSRP/SINR |
| FTP 速率低 | MCS 等级低 | 检查 CQI 上报 |
| FTP 中断 | TCP 拥塞/丢包 | 检查重传统计 |
| 上传失败 | UE 功率受限 | 检查 UE 发射功率 |

| 5 | UE3 附着 | 第三个 UE 附着 | IMSI=460001234567892 | CONNECTED |
| 6 | UE4 附着 | 第四个 UE 附着 | IMSI=460001234567893 | CONNECTED |

### 移动中业务测试
| 步骤 | 步骤名 | 操作描述 | 参数 | 预期结果 |
|------|--------|---------|------|---------|
| 1-3 | 环境重置、小区建立、UE 附着 | 同上 | - | - |

| 5 | UE 移动 | UE 从小区中心移动到边缘 | 从 (0,0) 到 (500,0) | 位置变化 |

| 7 | PING 结果分析 | 检查切换前后 PING 时延 | - | 切换时延增加<50ms, 丢包<2 个 |
